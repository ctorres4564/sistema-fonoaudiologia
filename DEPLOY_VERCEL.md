# 🚀 Guia de Deploy no Vercel

Este guia mostra como fazer o deploy do seu sistema de fonoaudiologia no Vercel de forma rápida e gratuita.

---

## 📋 Pré-requisitos

Antes de fazer o deploy, certifique-se de que:

* ✅ O Firebase Authentication está habilitado

* ✅ O Firestore Database está criado e configurado

* ✅ A aplicação está funcionando corretamente no ambiente local

* ✅ Você tem uma conta no GitHub (para versionar o código)

* ✅ Você tem uma conta no Vercel (gratuita)

---

## 🎯 Passo 1: Criar Repositório no GitHub

### Opção A: Usando a Interface do GitHub

1. Acesse [https://github.com/new](https://github.com/new)

2. Nomeie o repositório: `sistema-fonoaudiologia`

3. Deixe como **Private** (Privado) para proteger os dados

4. **NÃO** inicialize com README, .gitignore ou licença

5. Clique em **"Create repository"**

### Opção B: Via Terminal (se preferir)

```bash
# Navegue até a pasta do projeto
cd sistema-fonoaudiologia

# Inicializar git (se ainda não foi feito)
git init

# Adicionar origin remoto
git rem
```
